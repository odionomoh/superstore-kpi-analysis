# 📊 Superstore KPI Analysis

# 🧠 Project Overview

Data analysis enables businesses to transform raw data into meaningful insights that support strategic decision-making.

This project focuses on identifying a dataset, defining a business problem, and establishing key performance indicators (KPIs) that will guide data analysis and visualization.

Using the Global Superstore dataset, this project explores retail sales transactions to understand revenue generation, profitability, product performance, and regional sales distribution.

# 🎯 Problem Statement

Retail organizations generate large volumes of sales data daily. Without proper analysis, it becomes difficult to determine which products, regions, or customer segments contribute the most to overall business performance.

# The goal of this project is to analyze retail sales data to:

- Identify products generating the highest revenue

- Determine which regions contribute the most to total sales

- Evaluate profitability across product categories

- Understand customer purchasing patterns

The insights generated from this analysis can help businesses improve sales strategy and operational efficiency.

# 📂 Dataset Information

This project uses the Global Superstore Dataset, a widely used dataset for practicing data analytics and business intelligence.

# Dataset Source:
Kaggle

https://www.kaggle.com/datasets/apoorvaappz/global-super-store-dataset

The dataset contains retail transaction data including customer information, product categories, shipping details, and sales performance metrics.

# 📊 Key Performance Indicators (KPIs)

To evaluate sales performance and business efficiency, the following KPIs were defined:

# 💰 Total Revenue
Measures the total sales generated across all transactions.

Total Revenue = SUM(Sales)

# 📈 Total Profit
Represents the overall profit earned from all sales transactions.

Total Profit = SUM(Profit)

# 🌍 Sales by Region
Analyzes which geographic regions generate the highest revenue.

Regions in the dataset include:

- Africa

- Asia Pacific

- Europe

- US

- LATAM

This KPI helps identify strong and weak performing markets.

# 🛍 Sales by Product Category

Evaluates the revenue contribution from each product category.

Main categories include:

- Furniture

- Office Supplies

- Technology

This analysis helps determine which product groups drive business revenue.

# 📅 Monthly Sales Trend

Tracks sales performance across time to identify:

- Growth patterns

- Seasonal demand

- Sales fluctuations

This KPI helps businesses forecast future sales trends.

# 🧾 Average Order Value

Measures the average amount customers spend per order.

Average Order Value = Total Sales / Number of Orders

# 🏆 Top Selling Products

Identifies products generating the highest revenue within the dataset.

This KPI helps businesses optimize:

- Inventory management

- Product promotion

- Sales strategies
